# Scale

Scale transforms can be used to zoom elements in and out.

## `zoom-in-*`

<table>
<thead>
<tr>
<th>Class</th>
<th>Styles</th>
</tr>
</thead>
<tbody>
<tr>
<td>

`zoom-in`

</td>
<td>

```css
--tw-enter-scale: 0;
```

</td>
</tr>
<tr>
<td>

`zoom-in-<number>`

</td>
<td>

```css
--tw-enter-scale: <number>%;
```

</td>
</tr>
<tr>
<td>

`zoom-in-(<custom-property>)`

</td>
<td>

```css
--tw-enter-scale: var(<custom-property>);
```

</td>
</tr>
<tr>
<td>

`zoom-in-[<value>]`

</td>
<td>

```css
--tw-enter-scale: <value>;
```

</td>
</tr>
</tbody>
</table>
