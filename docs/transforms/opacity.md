# Opacity

While technically not a transform, opacity can be used to fade elements in and out.

## `fade-in-*`

<table>
<thead>
<tr>
<th>Class</th>
<th>Styles</th>
</tr>
</thead>
<tbody>
<tr>
<td>

`fade-in`

</td>
<td>

```css
--tw-enter-opacity: 0;
```

</td>
</tr>
<tr>
<td>

`fade-in-<number>`

</td>
<td>

```css
--tw-enter-opacity: calc(<number> / 100);
```

</td>
</tr>
<tr>
<td>

`fade-in-(<custom-property>)`

</td>
<td>

```css
--tw-enter-opacity: var(<custom-property>);
```

</td>
</tr>
</tr>
<tr>
<td>

`fade-in-[<value>]`

</td>
<td>

```css
--tw-enter-opacity: <value>;
```

</td>
</tr>
</tbody>
</table>
